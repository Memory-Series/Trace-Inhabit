# 拓跋玉儿 — 人物形象与设计

## 视觉设计

### 经典造型描述

拓跋玉儿是《轩辕剑·天之痕》三位主角中外观最为鲜亮的一位，以红色为主调，结合西域元素，展现出异族少女的泼辣与灵动。

#### 发型
- **黑色长发**：乌黑如瀑，长至腰部
- **双髻造型**：顶部两侧扎成两个发髻（丫鬟头），增添少女感
- **刘海**：轻薄齐眉刘海，露出额头

#### 面容
- **杏眼**：大而灵动，眼神中带英气
- **肤色**：白皙但不过分白皙，带有健康的红润
- **表情**：不笑时略显冷傲，笑时甜美可人

#### 服装
- **主色**：红色系（鲜红、朱红、暗红渐变）
- **款式**：胡服风格的紧身劲装，非传统汉服
- **上装**：红色抹胸+短褂，肩部有披帛装饰
- **下装**：配套红色长裤或裙子（便于行动）
- **腰间**：挂有小型药囊、葫芦、符咒袋
- **鞋子**：短靴或绣花鞋

#### 配饰
- **耳环**：简洁的金属耳环，异族风格
- **手镯**：左手腕有手镯或护腕
- **发饰**：发髻上簪有简单发饰（花朵或金属）

### 不同场景造型

#### 日常装
红色胡服，简洁利落，适合旅行和战斗。

#### 战斗装
束发紧身，披风扬起，体现女侠气势。

#### 后期/结局装
换装后更加成熟华丽（剧透，暂不详述）。

## 设计理念

拓跋玉儿的形象设计体现了以下几个核心概念：

1. **异族风情**：拓跋族是架空的古夏后裔，服装融合了西域和草原元素
2. **少女活力**：红色代表热烈与活力，与角色性格相符
3. **实用主义**：服装便于行动，体现其流浪武者的身份
4. **色彩对比**：与陈靖仇的蓝绿色、宁珂的紫白色形成对比

## 表情与神态

| 状态 | 描述 |
|------|------|
| 平静 | 略带冷淡，眼神警觉 |
| 微笑 | 甜美，但不会大笑 |
| 生气 | 眉头紧锁，眼神凌厉 |
| 害羞 | 脸红，目光闪躲 |
| 战斗 | 眼神坚定，气势逼人 |
| 温柔 | 声音放轻，眼神柔和 |
