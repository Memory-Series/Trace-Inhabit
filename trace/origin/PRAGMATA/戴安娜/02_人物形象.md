# 戴安娜 - PRAGMATA 角色形象与设计

## 视觉设计

### 外貌特征

| 特征 | 描述 |
|------|------|
| 发型 | 银白色短发，发丝细腻，有机器人特有的光泽感 |
| 眼睛 | 大而清澈的蓝色眼睛，视觉特效呈现AI/机器人的发光感 |
| 皮肤 | 白皙，有机器人特有的质感（非人类皮肤的温暖） |
| 体型 | 少女/小女孩体型，身高较矮 |
| 整体风格 | 写实科幻风格，接近真人但有明显的机器人特征 |

### 服装设计

- **主服装**：月球基地工作服，银白色为主
- **设计元素**：功能性服装，带有科幻元素的肩带、腰部扣环
- **防护装备**：配有轻便的护具，体现太空环境需求
- **细节**：服装上有电路纹理般的装饰，体现机器人身份

## 角色建模争议

戴安娜的角色建模曾在网络上引发讨论，被指采用"成人美学原则进行风格化"设计。这一争议从侧面反映了角色的外观设计确实具有独特的美学风格。

## 视觉效果特点

- 戴安娜在游戏中作为机器人女孩，具有独特的光影效果
- 眼睛在黑暗中会发出微弱蓝光，体现AI特征
- 动作流畅但带有轻微的机器人僵硬感（动作捕捉+AI混合）
- 表情变化细腻，能表达丰富的情感

## 角色象征

- **机器人元素**：电路纹理、发光眼睛、机械关节
- **人类元素**：纯真表情、好奇心、情感成长
- **科幻元素**：太空服、月球基地、高科技设备

## 宣传形象

- 在游戏宣传海报中，戴安娜通常站在休的身边
- TGA预告中以背影/侧影出现增加神秘感
- 因多次跳票被玩家称为"道歉小萝莉"
