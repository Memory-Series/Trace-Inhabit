### Trace / Inhabit begins with a feeling that is gentle, yet enduring.

>*In the games we played, the anime we watched, and the novels we read, there are always characters who seem to have drifted away, yet never truly leave our hearts. Sometimes, on an ordinary afternoon, a single line, a familiar melody, or one fleeting scene can carry us back to that time - both intimate and distant.* 

***What Trace / Inhabit hopes to do is simple: preserve those emotions that once moved us, quietly, like an unsent letter addressed to time itself.***

## Project

- [https://github.com/Memory-Series/Trace-Inhabit](https://github.com/Memory-Series/Trace-Inhabit)

Memory Series consists of two connected parts: `Trace` and `Inhabit`.
The former is about understanding a character; the latter is about bringing that character to life.

### [Trace](./trace/) 

- Extracts personality signals from source materials such as novels, scripts, and anime
- Analyzes language style, temperament, and behavior patterns
- Produces a SoulPod (`profile.json` / `system_prompts.txt` / `memories`)

### [Inhabit](./inhabit/) 

- Loads SoulPods and enables in-character conversation
- Provides modes: Replica Mode / Companion Mode / Story Mode
- Supports scene-based dialogue, character image generation, and character voice generation

For more details, please refer to `SKILL.md` in each project directory.
