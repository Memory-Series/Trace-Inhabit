# Memory Series（Trace / Inhabit）
### 寻迹 / 入心

### Trace / Inhabit 的起点，是一种很轻、却很长的惦念。

>*我们玩过的游戏、看过的动漫、读过的小说里，总有一些角色，明明已经走远，却始终留在心里。有时候只是一个普通的午后，一段台词、一首配乐、一个画面，就会把我们带回那段熟悉又陌生的时光。* 

***Trace / Inhabit 想做的，只是把那些曾经触动过我们的情绪，安静地保存下来，像给时间留一封不会寄出的信。***

## 项目简介

- [https://github.com/Memory-Series/Trace-Inhabit](https://github.com/Memory-Series/Trace-Inhabit)

Memory Series 由两个相互衔接的部分组成：`Trace（寻迹）` 与 `Inhabit（入心）`。
前者负责“理解角色”，后者负责“让角色活起来”。

### [Trace · 寻迹](./trace/) 

- 从小说、剧本、动漫等素材中提取角色人格线索
- 分析语言风格、性格特征与行为模式
- 生成 SoulPod（`profile.json` / `system_prompts.txt` / `memories`）

### [Inhabit · 入心](./inhabit/) 

- 加载 SoulPod，以角色身份进行对话
- 提供模式：复刻模式 / 伴侣模式 / 故事模式
- 支持能力：角色场景对话、角色图像生成、角色语音生成

更多信息请参考各项目目录下的 SKILL.md。
